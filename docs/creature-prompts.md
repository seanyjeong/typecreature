## 작업 팁

### Adobe Firefly 설정
- **Style**: Pixel art / 8-bit
- **Aspect Ratio**: 1:1 (정사각형)
- **Content Type**: Art

### 포토샵 후처리
1. 배경 제거 (투명 PNG)
2. 사이즈 통일 (64x64 또는 128x128)
3. 색상 보정 (희귀도별 톤 맞추기)
   - Legendary: 금빛/신비로운 글로우
   - Epic: 선명한 판타지 색상
   - Rare: 원소별 특징 색상
   - Common: 부드럽고 심플한 색상

### 파일 저장
```
TypingTamagotchi/Assets/Creatures/
├── legendary/
├── epic/
├── rare/
└── common/
```
