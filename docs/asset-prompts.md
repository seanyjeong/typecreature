## 사용 팁

1. **Midjourney**: `--ar 1:1 --style raw --v 6` 추가
2. **DALL-E**: 프롬프트 그대로 사용
3. **Stable Diffusion**: Negative prompt에 `realistic, 3d, detailed, complex` 추가

## 후처리

1. 64x64로 리사이즈
2. 배경 투명화
3. 파일명: `1.png` ~ `50.png` (크리처), `불꽃알.png` 등 (알)
4. 저장 위치: `Assets/Creatures/`, `Assets/Eggs/`
